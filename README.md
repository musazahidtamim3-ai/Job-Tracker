## Here is my answer to the related Questions :

## 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

#### ***Ans : we can access a element by it's id name using getElementById, we can access collection of elements by it's class name using getElementByClassName, we can find first matching element by querySelector and get a list of elements as a nodelist using querySelectorAll***

## 2. How do you create and insert a new element into the DOM?

#### Ans : ***We can create an element by use document.createElement("tag name") and insert a new element using parentNode.appendChild(newElemnet) into the DOM.***

## 3. What is Event Bubbling? And how does it work?

#### ***Ans : Event Bubbling is a javaScript process where if a event happen in child, it move up from his parent to grand parent and then document at last.***

## 4.  What is Event Delegation in JavaScript? Why is it useful?

#### ***Ans : Event Delegation is a process where we can use one eventlistener in an child's parent instead of using many event listener in  different diffrent childs. it is useful because => code become small, improves performance and reduce memory usage.***

## 5. What is the difference between preventDefault() and stopPropagation() methods?

#### ***Ans : PreventDefault() closes browser's default workflow. On the other hand, stopPropagation() closes event bubbling .***
